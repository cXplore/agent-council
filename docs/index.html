How it works

Connect

Point it at your project. If you already have agents, you're ready. If not, it scans your codebase and generates a team.

Meet

Ask Claude Code for a meeting in plain language. Agents deliberate in structured rounds — independent thinking first, then building on each other.

Watch

See every agent's response appear live. Add your own voice to the discussion. Review past meetings anytime.

The meeting system

Every meeting includes three mandatory roles: project-manager (what's real), critic (what's wrong), and north-star (what's possible). Plus domain agents relevant to the topic.

Round 1 — all agents respond independently, in parallel. No anchoring, no groupthink.

Round 2+ — agents read the full conversation and respond sequentially. They engage with each other, challenge assumptions, and build on ideas.

The meeting file is the hub. Everyone reads from it, everyone writes to it. You watch it build up live.

Get started

Desktop App

Download and run. No terminal needed.

Download for Windows

From Source

Clone, install, run. For developers.

git clone https://github.com/cXplore/agent-council
cd agent-council
npm install
npm run dev
# Opens at http://localhost:3003

MCP Integration

Optional two-way communication with Claude Code. The facilitator sends live progress events and checks for your input between rounds.

Claude Desktop

Add to claude_desktop_config.json

{
  "mcpServers": {
    "agent-council": {
      "command": "node",
      "args": ["/path/to/agent-council/mcp/server.mjs"]
    }
  }
}
